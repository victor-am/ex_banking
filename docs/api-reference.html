    ex_banking v0.1.0 API Reference

    Modules

    ExBanking

    The ExBanking module serves as an entrypoint for the application, it's APIs are the only things meant to be directly exposed to the outside world.

    ExBanking.Account

    The Account module is responsible for handling interactions between the ExBanking functions and the GenServer implemented in AccountServer.

    ExBanking.AccountServer

    The AccountServer module is a GenServer that is responsible for one user account at a time (multiple of them are spawned under the AccountsSupervisor).

    ExBanking.Money

    The functions inside this module are used to convert money between integer and 2 decimal places float formats.
